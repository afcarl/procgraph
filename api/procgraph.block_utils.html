textlog Module¶

class procgraph.block_utils.textlog.TextLog(name, config, library)[source]¶

Bases: procgraph.core.block.Generator

This represents a generic log reader that reads from a file line-by-line.

Subclasses should overload the parse_format() static method. Also they should put the adequate output() methods.
