check_rgb(block, input)

source code 
Checks that the selected input is either a RGB image. That is, a numpy array of uint8 of shape H x W x 3. Raises BadInput if it is not.

check_rgb_or_grayscale(block, input)

source code 
Checks that the selected input is either a grayscale or RGB image. That is, a numpy array of uint8 either H x W, H x W x 3, or HxWx4. Raises BadInput if it is not.
