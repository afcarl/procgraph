source code 
Provides a RGB representation of the values by interpolating the range [min(value),max(value)] into the colorspace [min_color, max_color].
Parameters:
  • value - The field to represent. :type value: array[HxW],H>0,W>0
  • max_value - If specified, everything above is clipped. :type max_value: float
  • min_value - If specified, everything below is clipped. :type min_value: float
  • min_color - Color to give to the minimum values. :type min_color: color
  • max_color - Color to give to the maximum values. :type max_color: color
  • nan_color - Color to give for regions of NaN and Inf. :type nan_color: color
Returns: array[HxWx3](uint8)
scale: A RGB image.
