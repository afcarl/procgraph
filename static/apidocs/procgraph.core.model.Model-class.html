Class Model

source code

A Model is a block and a generator.

cleanup() and finish(): "cleanup" is the emergency clean up. It is called after finish().

Order in which init(), update(), cleanup() can be called: - init(), update(), ..., update(), finish(), cleanup() - init(), finish(), cleanup() - init(), cleanup()
